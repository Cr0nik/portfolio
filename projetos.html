Projetos

Já fiz alguns projetos pessoais, que contém dois jogos: o Freeway (um jogo muito clássico) e o Pong. Ambos os jogos foram desenvolvidos totalmente em Java Script. Além dos jogos tenho outros projetos em Python que se encontram no meu GitHub

Comecei a estudar o ramo da programação em dezembro de 2022, porém vários dos meus projetos foram excluídos. Desde o início de junho de 2023, estou em uma escola de programação Full-Stack, e irei começar a divulgar todos os meus projetos por aqui e pelo Github

O jogo Freeway e o Pong eram parte de cursos sobre JavaScript na empresa Alura, em ambos eu fiz grandes mudanças na jogabilidade e no design, que me ajudaram a desenvolver uma lógica melhor de programação.
